# Instalación

## Requisitos

- [RustDesk](https://rustdesk.com/) instalado en el equipo local
- Acceso a un servidor API RustDesk (ver [Guía del servidor](../servidor/index.md))
- Node.js 18+ (solo para desarrollo)

## macOS

1. Descarga `RustDesk Manager-x.x.x-universal.dmg` de [Releases](https://github.com/abdatabase/rustdesk-manager/releases)
2. Abre el DMG y arrastra la app a Aplicaciones
3. Abre RustDesk Manager

!!! note "Primera ejecución"
    macOS puede mostrar un aviso de seguridad. Ve a **Preferencias del Sistema > Seguridad y Privacidad > General** y haz clic en **"Abrir igualmente"**.

## Windows

1. Descarga `RustDesk Manager Setup x.x.x.exe` de [Releases](https://github.com/abdatabase/rustdesk-manager/releases)
2. Ejecuta el instalador
3. Abre RustDesk Manager

!!! note "Windows SmartScreen"
    Si Windows SmartScreen bloquea el instalador, haz clic en **"Más información"** > **"Ejecutar de todos modos"**.

## Siguiente paso

Una vez instalada la app, continúa con la [Configuración inicial](configuracion.md).
